🚚 Backlog / DocLinks migration tool

Trang này dùng để chuyển data backlog-* + docLinks-* từ localStorage giữa các domain (VD Vercel → Zingplay).

① Trên Vercel — Export

Bấm nút bên dưới khi Anh đang ở trang ctp1-task-manager-web.vercel.app/migrate.html. File JSON sẽ tự download.

② Trên Zingplay — Import

Bấm nút bên dưới khi đang ở taskctp.zingplay.dev/migrate.html. Chọn file JSON vừa export.

Sau khi import xong

  1. Đóng trang này
  2. Vào /projects/<CTP1> → Backlog/DocLinks sẽ thấy data từ localStorage
  3. App tự upload data lên DB → reload lại lần nữa thấy data hiển thị từ DB
  4. Từ giờ data ở DB, cross-device, team thấy chung
